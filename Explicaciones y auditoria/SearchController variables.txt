$searchService
    TIPO: SearchService (inyectado)
    PROPÓSITO: Servicio que realiza la búsqueda real
    FLUJO: Controller → Service → API
    DONDE SE USA: En método search() para llamar a la API

$request
    TIPO: Illuminate\Http\Request
    PROPÓSITO: Contiene los datos del formulario de búsqueda
    CONTENIDO: query, headers, método HTTP, etc.
    DONDE SE USA: En search() para obtener el término de búsqueda

$query
    TIPO: string
    PROPÓSITO: Término que el usuario quiere buscar
    EJEMPLOS: "jacket", "phone", "electronics"
    ORIGEN: $request->input('query')
    FLUJO: Formulario → Request → Controller → Service

$resultados
    TIPO: array
    PROPÓSITO: Resultados de la búsqueda organizados por tienda
    ORIGEN: Retorno de $searchService->search($query)
    FLUJO: Service → Controller → Vista

$total_products
    TIPO: int
    PROPÓSITO: Número total de productos encontrados en todas las tiendas
    CÁLCULO: Suma de count() de cada array de tienda
    USO: Para mostrar estadísticas al usuario

$total_stores
    TIPO: int
    PROPÓSITO: Número de tiendas que tienen resultados
    CÁLCULO: count($resultados)
    EJEMPLO: Si hay 4 claves en $resultados → total_stores = 4
    USO: Para mostrar estadísticas al usuario
